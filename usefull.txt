Чтобы подчеркнуть текст, вы можете использовать <u> Tag </u>  тег

Чтобы зачеркнуть текст, когда горизонтальная линия пересекает символы, вы можете использовать <s> Tag </>тег.

Вы можете использовать <hr> Tag </hr> тег, чтобы добавить горизонтальную линию по ширине содержащего его элемента.

Чтобы анимировать элемент, вам необходимо знать свойства анимации и @keyframesправило. 
animation-nameустанавливает имя анимации, которое позже используется, @keyframesчтобы сообщить CSS, какие правила соответствуют каким анимациям.

Существует несколько вариантов размещения гибких элементов вдоль линии, являющейся главной осью. Один из наиболее часто используемых - justify-content: center;выравнивание всех гибких элементов по центру внутри гибкого контейнера. Другие варианты включают:

flex-start: выравнивает элементы по началу гибкого контейнера. Для строки это перемещает элементы слева от контейнера. Для столбца это перемещает элементы в верхнюю часть контейнера. Если не justify-contentуказано иное, это выравнивание по умолчанию .
flex-end: выравнивает элементы по концу гибкого контейнера. Для строки это перемещает элементы справа от контейнера. Для столбца это перемещает элементы в нижнюю часть контейнера.
space-between: выравнивает элементы по центру главной оси, добавляя дополнительное пространство между элементами. Первый и последний элементы подталкиваются к самому краю гибкого контейнера. Например, в ряду первый элемент находится напротив левой стороны контейнера, последний элемент - напротив правой стороны контейнера, а оставшееся пространство равномерно распределяется между другими элементами.
space-around: аналогично, space-betweenно первый и последний элементы не привязаны к краям контейнера, пространство распределяется вокруг всех элементов с половиной пространства на обоих концах гибкого контейнера.
space-evenly: Равномерно распределяет пространство между гибкими элементами с полным пространством на обоих концах гибкого контейнера.
